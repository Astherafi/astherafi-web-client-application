## About

Astherafi is a comprehensive DeFi utilities platform providing powerful blockchain tools for traders, developers, and crypto enthusiasts. Built with modern web technologies and powered by AI.

## Features

### ASTHER DEX
Real-time Solana token discovery with trending tokens and new listings powered by DexScreener. Integrated with Jupiter for seamless token swaps.

### Token Analyzer
Multi-chain token security analysis supporting Solana, Ethereum, BSC, Polygon, Arbitrum, Base, Avalanche, Fantom, and Optimism. Powered by GoPlus Labs security data.

### Staking Platform
Multi-pool ASTHER token staking system with three pool options:
- **Flexible Pool** - 60% APY, no lock period
- **3-Day Pool** - 90% APY, 3-day lock
- **7-Day Pool** - 150% APY, 7-day lock

### Asthera AI
AI-powered chat assistant for DeFi guidance, token analysis, and blockchain education.

### Asthera Free API
Free blockchain API endpoints for developers to integrate token data and security checks into their applications.

### Password Security Tool
Advanced password strength analyzer with real-time scoring and security recommendations.

### BSC Bundler
Bundled transaction tool for Binance Smart Chain operations.

## Tech Stack

- **Frontend**: React, Tailwind CSS, Vite
- **Backend**: Node.js, Express
- **APIs**: DexScreener, GoPlus Labs, Jupiter, Helius
- **AI**: DeepSeek Integration
- **Blockchain**: Solana, BSC (BEP-20)

Available in English, Chinese, and Arabic.
